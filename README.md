# 🧘 Kegel Exercise Timer · 凯格尔运动计时器

[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

一个简洁、优雅的交互式计时器，专为凯格尔（Kegel）运动设计。支持**中英文双语切换**，可自定义紧绷/放松时长与循环组数，帮助您科学、规律地完成训练。

**在线体验**：👉 [点击这里](https://clark-mu.github.io/kegel_trainer/)

---

## ✨ 功能特点

- **⏱️ 精准计时**：严格按照“紧绷 → 放松”交替循环，每个阶段倒计时清晰可见。
- **🔄 完全自定义**：通过设置面板自由调整：
  - 紧绷时长（默认 10 秒）
  - 放松时长（默认 5 秒）
  - 循环组数（默认 15 组）
- **🌐 多语言支持**：一键切换中文 / 英文，所有界面文字同步翻译。
- **🎵 声音反馈**：阶段切换和训练完成时伴有提示音，无需一直盯着屏幕。
- **🎨 视觉交互**：
  - 环形进度条直观显示当前阶段剩余时间
  - 阶段颜色区分（红色紧绷 / 青色放松）
  - 完成时播放庆祝动画（并伴有彩蛋效果）
- **⌨️ 键盘快捷键**：
  - 空格键：开始 / 暂停
  - `R` 键：重置计时器
- **📱 响应式设计**：在手机、平板、桌面设备上均可流畅使用。

## 🚀 如何使用

### 在线使用（推荐）
直接访问 GitHub Pages 部署的地址（见上方链接），无需安装任何软件。

### 本地运行
1. 克隆本仓库或下载 `index.html` 文件。
2. 双击 `index.html` 在浏览器中打开即可使用。

### 训练步骤
1. 点击 **“开始”** 按钮（或按空格键）启动计时。
2. 跟随提示进行 **紧绷** 和 **放松** 交替训练。
3. 每完成一组，进度会自动更新；全部完成后会有庆祝音效。
4. 随时可点击 **暂停/继续** 或 **重置**。

---

## ⚙️ 自定义设置

点击右上角的 **⚙️** 齿轮图标打开设置面板，您可以分别调整：

| 参数   | 说明           | 范围    | 默认值 |
|--------|----------------|---------|--------|
| 紧绷   | 每次收紧时长   | 1–120秒 | 10秒   |
| 放松   | 每次放松时长   | 1–120秒 | 5秒    |
| 循环   | 完整训练组数   | 1–99组  | 15组   |

调整后点击 **“应用”** 即可生效，计时器会自动重置。点击 **“恢复默认”** 可一键回到医生建议的经典参数。

- 原生 HTML5 + CSS3 + JavaScript（ES6）

- 响应式设计（Flexbox + 媒体查询）
